# Personal Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Smooth navigation
- Mobile-friendly hamburger menu
- Multiple sections (Home, About, Projects, Contact)
- Clean and modern UI

## Files

- `abhi.html` - Main HTML structure
- `abhi.css` - Styling and responsive design
- `abhi.js` - Interactive features and smooth scrolling
- `abhi.jpeg` - Profile image

## Deployment on Netlify

### Option 1: Direct Upload
1. Go to [Netlify](https://netlify.com)
2. Sign up or log in
3. Drag and drop the entire portfolio folder
4. Your site will be deployed instantly

### Option 2: Git Integration
1. Push this repository to GitHub
2. Connect your GitHub account to Netlify
3. Select this repository
4. Click "Deploy"
5. Your site will auto-deploy on every push

### Option 3: Netlify CLI
```bash
npm install -g netlify-cli
netlify deploy --prod
```

## Requirements

**No dependencies required!** This is a pure static site that doesn't need:
- Node.js
- npm packages
- Build process
- Server-side runtime

Just open `abhi.html` in a browser to view locally, or deploy to Netlify for hosting.

## Local Development

Simply open `abhi.html` in your web browser or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server
```

Then navigate to `http://localhost:8000` or `http://localhost:8080`
